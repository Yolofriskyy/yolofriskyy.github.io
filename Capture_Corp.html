Multiplayer Systems

Lobby setup served as a basic system for players to quickly customise and create their lobbies to their liking, i.e. setting a player limit. Lobby joining provides the player with functionality to choose any setup lobby. Lobbies refresh every few seconds.

This system uses the Steamworks SDK framework "Matchmaking and Lobbies", which allows hosts to share their local network information with other players through Steamâ€™s network infrastructure. Using Steam's NAT traversal techniques, players from all over the world that use Steam can connect to lobbies created by other players. The lobby host acts as the server, while other players connect as clients.
